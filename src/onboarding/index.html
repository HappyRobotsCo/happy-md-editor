Welcome to Happy MD Editor

View, edit, and save markdown files — right from your browser.

Step 1: Enable File URL Access

To open local .md files directly in the browser, you need to grant file URL access:

  1. Copy this URL and paste it into your address bar: chrome://extensions
  2. Find Happy MD Editor in the extensions list
  3. Click Details
  4. Toggle "Allow access to file URLs" to ON

Note: chrome:// URLs cannot be linked directly — you must copy and paste.

Step 2: Open the Editor

Click the Happy MD Editor icon in your browser toolbar to open a new editor tab.

You can also drag and drop any .md file onto the editor to start editing.

Features

  • WYSIWYG Editing Rich text editing with live formatting — no markdown syntax needed
  • Source Mode Edit raw markdown with syntax highlighting and a formatting toolbar
  • Save to Disk Save changes directly back to local files using the File System Access API
  • Themes Light and dark themes that follow your OS preference
  • Keyboard Shortcuts Familiar shortcuts for bold, italic, headings, and more
